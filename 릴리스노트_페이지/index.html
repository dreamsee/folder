MCP

Revamped tool picker experience

We've totally revamped the tool picker this iteration and adopted a new component called Quick Tree to display all the tools.

The new tool picker provides better search capabilities, hierarchical organization, and improved performance when dealing with hundreds of tools.

// Example tool configuration
{
  "tools": {
    "enabled": true,
    "picker": {
      "style": "tree",
      "search": "fuzzy"
    }
  }
}

Enable more than 128 tools per agent request

Previously, agents were limited to 128 tools per request. We've now implemented a batching system that allows unlimited tools.

Chat

GPT-5 availability

Starting today, GPT-5 is rolling out to all paid GitHub Copilot plans. GPT-5 is OpenAI's most capable model yet, bringing new advances in reasoning, coding, and chat.

Learn more about the GPT-5 model availability in the GitHub Changelog.

Open the Chat view and choose GPT-5 from the model picker to start using it for your chat conversations in VS Code.

Chat checkpoints

Setting: chat.checkpoints.enabled

We've introduced checkpoints that enable you to restore different states of your chat conversations. You can easily revert edits and go back to previous points in your conversation history.

// Enable chat checkpoints
"chat.checkpoints.enabled": true

Productivity

Check out multiple branches simultaneously with Git worktrees

Git worktrees allow you to have multiple branches checked out at the same time in different folders, making it easier to work on multiple features or bug fixes simultaneously.

  • Create new worktrees from the Source Control view
  • Switch between worktrees quickly
  • Manage all your worktrees from a single VS Code window

Manage coding agent sessions in a dedicated view

A new dedicated view for managing your coding agent sessions provides better organization and control over your AI-assisted development workflow.
